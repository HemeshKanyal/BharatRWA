# BharatRWA 🇮🇳
### The Premier Real-World Asset (RWA) Tokenization & Compliance Platform

BharatRWA is a state-of-the-art decentralized platform designed to tokenize Real-World Assets (RWAs) while maintaining strict regulatory compliance through privacy-preserving Zero-Knowledge Proofs (ZK-KYC).

![BharatRWA Dashboard](https://raw.githubusercontent.com/HemeshKanyal/BharatRWA/main/frontend/public/dashboard-preview.png) *

## Key Features

- **Asset Fractionalization**: Convert physical assets like Gold, Silver, Real Estate, and Commodities into liquid ERC-20 tokens.
- **ZK-KYC Compliance**: Privacy-first identity verification using **Noir ZK-Circuits**. Prove you are over 18 or from a specific region without revealing your identity.
- **On-Chain Governance & Registry**: A transparent, immutable registry of all tokenized assets and their custodians.
- **Synthetic Exchange**: A high-performance trading interface with real-time market data, candle charts, and order books.
- **Automated Dividends**: Distribute revenue generated by underlying assets directly to token holders.

## Architecture

The project is divided into four main components:

### 1. Smart Contracts (`/bharat-rwa`)
Built using the **Foundry** framework, these contracts manage the core logic of the ecosystem:
- `AssetRegistry.sol`: The source of truth for all tokenized assets.
- `ComplianceManager.sol`: Handles ZK-KYC verification and whitelist management.
- `BharatRWAToken.sol`: Customizable ERC-20 tokens representing fractional ownership.
- `AssetOracle.sol`: Integrates off-chain price feeds for accurate asset valuation.

### 2. ZK-KYC Circuits (`/zk_kyc`)
Developed in **Noir**, the high-level language for ZK-proofs:
- Validates age, document authenticity, and sanction status.
- Generates UltraPlonk proofs that are verified on-chain by the `ZKVerifier.sol` contract.

### 3. Backend (`/backend`)
A Node.js/Express server that acts as the bridge between the ZK circuits and the frontend:
- **ZK Prover Service**: Generates ZK proofs for users using `nargo` or `bb`.
- **Market Data Engine**: Aggregates prices from external APIs (Binance, CoinGecko) and provides synthetic market history for the frontend charts.
- **Transaction Relayer**: Facilitates buy/sell operations on the Sepolia testnet.

### 4. Frontend (`/frontend`)
A premium **Next.js** application providing a seamless user experience:
- Real-time trading dashboard with **Lightweight Charts**.
- KYC onboarding flow with ZK-proof generation.
- Asset management and portfolio tracking.

## Getting Started

### Prerequisites
- Node.js v18+
- [Foundry](https://getfoundry.sh/) (for contracts)
- [Nargo](https://noir-lang.org/docs/getting_started/installation) (for ZK circuits)
- Metamask or any EIP-1193 wallet

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/HemeshKanyal/BharatRWA.git
   cd BharatRWA
   ```

2. **Setup Smart Contracts**
   ```bash
   cd bharat-rwa
   forge build
   ```

3. **Setup Backend**
   ```bash
   cd ../backend
   npm install
   # Create .env with PRIVATE_KEY and SEPOLIA_RPC_URL
   node server.js
   ```

4. **Setup Frontend**
   ```bash
   cd ../frontend
   npm install
   npm run dev
   ```
